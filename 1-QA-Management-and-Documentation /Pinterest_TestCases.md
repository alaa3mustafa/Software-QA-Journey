## Test Case Details

* **Test Case ID:** CPPicture-01

* **Description:** Verify profile picture change in the Pinterest application (Positive Scenario).

* **Pre-conditions:**

  * The user is successfully logged into the application.

  * A stable internet connection is established.

## Test Steps

1. Open the Pinterest application.

2. Tap on the **Profile icon** located at the bottom right of the screen (to open the profile page and settings).

3. Tap on the **Profile picture** at the top left of the screen (to view the profile interface as seen by visitors).

4. Click on the **"Edit Profile"** button located below the profile description.

5. Click on the **"Edit"** option underneath the profile picture (to trigger the change pop-up).

6. Select **"Update profile picture"** and choose an image from the device's gallery.

7. Click **"Done"** (top right) twice consecutively to save the updates.

## Expected Result

The profile picture is updated successfully, and the new image is immediately displayed in the user interface.

## Actual Result

**Pass**
